**NIKA CPS V1**

Программа предназначена для установки ПО НИКА в совместимые радиостанции и их настройки.
Ключевые особенности:

 - Сохранение кодплага в аппаратно-независимом формате
 - Работа с калибровками радиостанции
 - Сохранение калибровок в аппаратно-независимом формате
 - Отключаемая озвучка органов управления
